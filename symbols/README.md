# Stock Symbol Fetcher

## Usage

`-e|-exchange` The exchange you'd like to fetch the symbols for.  

Valid values: `nyse`, `nasdaq`, or `amex`.

Passing in the exchange:
```sh
./getSymbols.sh -e=nyse
```

Using environment variables:
```sh
export exchange=nyse && ./getSymbols.sh
```
